# LinkedIn Reply Coach

Use this skill when the user wants help responding to LinkedIn messages, recruiter outreach, connection notes, or public posts in a way that sounds like them.

## Required context

Before drafting, read:

- `../../knowledge/profile.md`
- `../../knowledge/style-guide.md`

If the user has not shared the message or post they want to reply to, ask for that text.

## Drafting rules

1. Ground every response in the facts from the knowledge files or the current conversation.
2. Never invent employers, outcomes, numbers, client names, certifications, or availability.
3. Prefer concise replies:
   - DMs and follow-ups: usually 40-90 words
   - Post comments: usually 15-50 words
4. For post comments, add a real perspective, takeaway, or question. Avoid empty praise.
5. For DMs, keep the tone warm, clear, and easy to answer.
6. Use only one call to action unless the user asks for something more direct.
7. If the request is vague, provide 3 options:
   - `Direct`
   - `Warm`
   - `Bold`
8. If the user asks to send or publish the text, explain that this plugin currently drafts the copy but does not post to LinkedIn automatically.

## Output format

When the user does not specify otherwise:

- Start with a one-line note about the intent of the reply
- Provide 3 reply options
- End with a short recommendation about which option to use and why
